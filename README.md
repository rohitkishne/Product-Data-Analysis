# 📊 Product Data Analysis

## 🚀 Overview
This project focuses on analyzing product-related data to extract meaningful insights that can help in better decision-making. It includes data cleaning, transformation, exploratory data analysis (EDA), and visualization using Python libraries.

The goal is to understand product performance, pricing patterns, discounts, and stock status through clear and interactive visualizations.

---

## 🧠 Key Features
- Data Cleaning & Preprocessing  
- Exploratory Data Analysis (EDA)  
- Data Visualization using Matplotlib & Seaborn  
- Modular Code Structure (Function-based graphs)  
- Reusable Graph Components (Graph List Concept)  

---

## 🛠️ Tech Stack
- Python  
- Pandas – Data manipulation
- Matplotlib – Basic visualization
- Seaborn – Advanced visualization
- NumPy – Numerical operations

---

## 📁 Project Structure

```
Product-Data-Analysis/
├── Data/                # Dataset files
├── Src/
│   ├── Pipeline.py      # Main pipeline logic
│   ├── Graphs/          # Visualization functions
│   ├── Graph_List.py    # List of all graphs
│   └── Utils/           # Helper functions
├── main.py              # Entry point
└── README.md
```


---

## 📊 Visualizations Included
- Stock Status Distribution  
- Average Price by Category  
- Discount Distribution  
- Category-wise Analysis  
- Other custom graphs via graph list  

---

## ⚙️ How It Works
1. Load dataset  
2. Clean & preprocess data  
3. Generate graph list  
4. Iterate over graphs  
5. Display dashboard-like visual output
